# Les zamālatain de Yarmūk

Lors de la bataille de **Yarmūk** (15 H / 636 EC), [[Ibn 'Amr ibn al-'Āṣ|'Abd Allāh ibn 'Amr]] récupère **deux zamālatain** (charges de chameau) de **manuscrits scripturaires** (كتب أهل الكتاب / كتب المتقدمين).

## Le fait

Ibn Kathīr (al-Bidāya 2/107) : « أصاب يوم اليرموك زاملتين من كتب المتقدمين، فكان يحدث منهما » — le fait matériel n'est contesté par personne, y compris les défenseurs de la Sunna.

## La controverse

- **Bishr al-Marīsī** (m. 218 H) : premier à accuser Ibn 'Amr d'avoir présenté le contenu des zamālatain comme des hadiths prophétiques. « وكان يقال له: لا تحدثنا عن الزاملتين » — « On lui disait : Ne nous raconte pas [ce qui vient] des deux zamālatain ! »
- **Maḥmūd Abū Rayya** (XXe s.) : reprend l'accusation dans Aḍwā' 'alā al-Sunna al-Muḥammadiyya
- Les défenseurs répondent qu'Ibn 'Amr distinguait ses sources (prophétiques vs scripturaires)

## Le contenu — ce que transportaient les charges

Deux charges de chameau de « kutub ahl al-kitāb » / « kutub al-mutaqaddimīn » — des manuscrits scripturaires judéo-chrétiens. La tradition reconnaît qu'Ibn ʿAmr **transmettait depuis ces manuscrits** (« fa-kāna yuḥaddithu minhumā » — Ibn Kathīr). Le contenu exact est inconnu, mais le matériel qui circule sous le nom d'Ibn ʿAmr dans le tafsīr et les faḍāʾil donne un aperçu : cosmogonie, eschatologie, géographie sacrée, récits des prophètes antérieurs.

## La double source d'Ibn ʿAmr

Les zamālatain ne sont pas la seule source scripturaire d'Ibn ʿAmr. Il a aussi étudié avec **[[Sarj al-Yarmūkī]]** — un maître **kitābī** (chrétien ou juif) oral. Les deux forment un système complet : la source **écrite** (zamālatain) et la source **orale** (Sarj).

```
SOURCES SCRIPTURAIRES
        │
   ┌────┴────────┐
   │             │
Zamālatain    Sarj al-Yarmūkī
(manuscrits)  (maître oral)
   │             │
   └────┬────────┘
        │
  IBN 'AMR IBN AL-'ĀṢ
        │
   ┌────┼────────────┐
   │    │            │
Nawf  Ka'b         Abū Hurayra
   │  (rencontre     │
   │   à La Mecque)  │
   │                 │
 Shahr              Hammām → ṣaḥīfa
   │
 Qatāda → recueils canoniques
```

Le matériel sort par **trois canaux** :
1. **Le canal Ka'b** — Ka'b rencontre Ibn ʿAmr à La Mecque, les deux partagent du matériel scripturaire. Le contenu circule ensuite via [[Nawf al-Bikālī|Nawf]] → [[Shahr ibn Ḥawshab|Shahr]] → [[Qatāda ibn Di'āma|Qatāda]].
2. **Le canal Abū Hurayra** — Abū Hurayra transmet d'Ibn ʿAmr (Mustadrak n° 8558, [[Traditions/Hijra vers le Shām|hijra vers le Shām]]). Le matériel est fixé dans la [[Personnes/Hammām ibn Munabbih|ṣaḥīfa de Hammām]].
3. **Le canal direct** — des traditions du tafsīr sont transmises directement depuis Ibn ʿAmr via des tābiʿīn syriens ([[Rajā' ibn Ḥaywa|Rajā']] → [[Personnes/Makhūl al-Shāmī|Makhūl]] → [[Al-Awzā'ī]]).

## Le paradoxe d'Ibn ʿAmr

La tradition reconnaît **deux faits simultanés** :
1. Ibn ʿAmr possédait des manuscrits scripturaires et transmettait depuis eux
2. Ibn ʿAmr est un Compagnon dont les hadiths sont acceptés dans les Ṣaḥīḥayn

Le système ne peut pas avoir les deux : si Ibn ʿAmr mélange matériel scripturaire et paroles prophétiques (ce que Bishr al-Marīsī affirme), alors ses hadiths sont contaminés. Si Ibn ʿAmr distingue rigoureusement ses sources (ce que les défenseurs affirment), alors pourquoi dit-on « لا تحدثنا عن الزاملتين » — « ne nous raconte pas [ce qui vient] des zamālatain » ? L'avertissement lui-même prouve que le mélange était perçu comme un risque réel.
