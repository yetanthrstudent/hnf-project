# Tārīkh Baghdād (تاريخ بغداد)

Équivalent baghdadien du *[[Tarikh Dimashq|Tārīkh Dimashq]]* d'Ibn ʿAsākir. Contient les biographies de tous les savants passés par Baghdād, avec leurs chaînes de transmission et les jugements critiques les concernant.

## Statut dans la tradition

Al-Khaṭīb est aussi l'auteur d'ouvrages méthodologiques fondateurs (*al-Kifāya*, *al-Jāmiʿ*). Al-Dhahabī le considère comme le fondateur de la science du hadith sous sa forme systématique. Le *Tārīkh Baghdād* est la source principale pour les transmetteurs irakiens — là où le *Tārīkh Dimashq* couvre les Shāmīs.

## Utilisation dans notre recherche

Biographies des transmetteurs irakiens, chaînes de transmission passant par Baghdād. Essentiel pour documenter les circuits de transmission irakiens et les comparer aux circuits shāmīs.
